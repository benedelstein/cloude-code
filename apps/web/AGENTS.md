# Cloude Code Web Client

This is a Next.js web client. It uses:
- Next.js app router
- TypeScript
- Tailwind CSS
- shadcn/ui
- AI SDK
- Cloudflare Agents SDK

## Important guidelines

- Before changing styling, layout, components, or responsive behavior, read the repo-root `docs/frontend/styling.md`.
- After completing visual changes, use a relevant browser tool to validate the affected UI and capture a screenshot when useful.
